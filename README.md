<!-- GETTING STARTED -->
## Como Utilizar

Siga os passos abaixo para garantir que a instalação ocorra corretamente.


### Instalação

1. Clone o repositório
   ```sh
   git clone https://github.com/anthonyreis/TesteCienciaDados.git
   ```
2. Instale as bibliotecas
* xldr
   ```sh
   pip install xldr
   ```
* scikit-learn
   ```sh
   pip install sklearn
   ```
* Pandas
   ```sh
   pip install pandas
   ```
